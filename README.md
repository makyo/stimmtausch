# Stimmtausch

Stimmtausch (`stimmtausch`) is a terminal-based MUCK client written in Go.

## Installing

```bash
sudo snap install --channel=edge stimmtausch
```

## More information

You can find out more on [stimmtausch.com](https://stimmtausch.com).
